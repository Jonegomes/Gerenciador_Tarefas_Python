# Gerenciador_Tarefas_Python_01

#### Descrição:
Projeto desenvolvido como prática de fundamentos em Python para análise de dados e automação de rotinas.
A aplicação simula uma lista de tarefas inteligente, que permite adicionar, listar, editar e excluir tarefas com diferentes níveis de prioridade.
Ao longo do projeto, trabalhei desde a entrada e saída de dados com input() e print(), passando por uso de operadores lógicos e aritméticos, estruturas condicionais (if, else, elif), loops (for, while), listas e dicionários, até a organização do código em funções.
O objetivo foi aprender a estruturar um programa do zero, praticando boas práticas como modularização, legibilidade e tratamento de erros simples.

#### Insights pessoais:
Esse projeto me ensinou a pensar como um programador: dividir problemas em partes menores, testar hipóteses e transformar lógica em solução. Foi o primeiro passo real para entender como o Python pode ser aplicado na análise de dados e no dia a dia profissional.
